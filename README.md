# regex


最近学习编译原理，基于NFA实现了正则表达式，代码刚写完，后续具体实现会出一篇博客放在 [https://xindoo.blog.csdn.net/](https://xindoo.blog.csdn.net/)，敬请关注。

目前还是demo，算是刚把引擎的骨架搭建起来，目前只支持 ` . ? * + () |`规则，后续继续完善代码。  

## 是什么不是什么？ 
这个引擎不会是一个可以用在生产环境的项目，但会是一个了解正则引擎背后工作原理的项目。
